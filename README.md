# Build My Idea - Landing Page

A single-page landing site inspired by [buildmyidea.com](https://buildmyidea.com/).

## Run It

Just open `index.html` in any browser:

```bash
open index.html
```

Or serve it locally:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

No build tools or dependencies required. Google Fonts are loaded via CDN.
